# Brotato Winter Mod
Bring the cold to your Brotato run all year round! This mod adds winter and holiday themed characters, items, and even a new map to the game!

## Added Content:
- 2+ new characters
    - Santa Claus
    - The Yeti
    - Toymaker
- Winter themed map
    - New themed visuals

[**Download mod here.**](https://youtube.com/)


## Requirements
- [ModLoader](https://github.com/GodotModding/godot-mod-loader)
- [ContentLoader](https://github.com/BrotatoMods/Brotato-ContentLoader)
- [ModOptions](https://github.com/BrotatoMods/Brotato-Mod-Options)
